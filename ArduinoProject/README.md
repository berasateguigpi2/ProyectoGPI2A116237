# Configure Makefile
    Para compilar y desplegar el proyecto en local cambiar las siguientes variables en 
    **src/FooProject/Makefile**:
	- PROJECT_DIR : Directorio donde has clonado el proyecto
	- ARDMK_DIR: Directorio de el Arduino-Makefile (en caso necesario clonar de
			 https://github.com/leka/Arduino-Makefile)
	- ARDUINO_DIR: Directorio de Arduino
